# Мониторинг цен: Москва → Венеция (обновление каждые 8 часов)

Скрипт раз в 8 часов тянет самую дешёвую цену по маршрутам Москва → Венеция (VCE)
и Москва → Тревизо (TSF), копит историю в `data/history.csv` и пересобирает
страницу `docs/index.html` с графиком и таблицей. Всё крутится бесплатно на
GitHub Actions + GitHub Pages — сервер не нужен.

## Что это НЕ делает (честно)
- Это **не** живые котировки в реальном времени. Travelpayouts Data API отдаёт
  цены из кеша поисков Aviasales за последние ~48 ч. Это надёжный ориентир по
  тренду «дорожает/дешевеет», но перед покупкой цену надо сверить на aviasales.ru
  по ссылке из карточки.
- Прямых рейсов Москва → Венеция сейчас нет — все варианты с пересадкой
  (обычно Белград/Air Serbia или Ереван).

## Запуск за 5 шагов

1. **Получи бесплатный токен.** Зарегистрируйся на travelpayouts.com →
   в профиле раздел «API token» скопируй токен. (Регистрация бесплатная,
   партнёрская программа Aviasales.)

2. **Создай репозиторий** на GitHub (можно приватный) и залей туда эти файлы:
   `monitor.py`, `.github/workflows/monitor.yml`, этот `README.md`.

3. **Добавь токен как секрет:** в репозитории → Settings → Secrets and variables
   → Actions → New repository secret. Имя: `TRAVELPAYOUTS_TOKEN`, значение — твой токен.

4. **Включи GitHub Pages:** Settings → Pages → Source = «GitHub Actions».

5. **Запусти вручную первый раз:** вкладка Actions → workflow «fare-monitor»
   → «Run workflow». Дальше он сам будет запускаться каждые 8 часов
   (00:00 / 08:00 / 16:00 UTC).

Страница появится по адресу `https://<твой-логин>.github.io/<репозиторий>/`.

## Настройка под себя

В начале `monitor.py`:
- `ROUTES` — маршруты. Добавь/убери направления (IATA-коды городов).
- Хочешь следить за конкретной датой вылета, а не «ближайший дешёвый» —
  скажи мне, добавлю фильтр по `departure_at` в `fetch_cheapest()`.
- `CURRENCY` — валюта (по умолчанию `rub`).

Периодичность меняется в `.github/workflows/monitor.yml`, строка `cron`.
Сейчас `0 */8 * * *` = каждые 8 часов. Например, каждые 4 часа: `0 */4 * * *`.

## Локальный тест
```bash
export TRAVELPAYOUTS_TOKEN=твой_токен
python monitor.py
# затем открой docs/index.html в браузере
```
